## Getting Started
1. Clone the repository:
   ```bash
   git clone https://github.com/Satya1998-debug/robotdog_guide.git
   ```
2. Navigate to the project directory:
   ```bash
    cd robotdog_guide
   ```
3. Intall uv library:
   ```bash
   curl -LsSf https://astral.sh/uv/install.sh | sh
   ```  
4. Sync the uv dependencies:
   ```bash
   uv sync
   ```
